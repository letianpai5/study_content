# git add xxx
> git add命令是告诉Git，把文件提交到仓库

# git commit
- git commit -m "xxxx"
> 该命令告诉Git，把文件提交到仓库
> 
> -m 后面输入的是本次提交的说明，可以输入任意内容（最好有意义）
> 该命令执行成功后会告诉：
>  - 1file changed :1个文件被改动
>  - 2 insertions : 插入了两行内容

# git status
> 该命令可以让我们时刻掌握仓库当前的状态
> 要随时掌握工作区状态使用这个

# git diff
> 该命令可以查看difference，显示的格式正是Unix通用的diff格式
> 如果git status告诉你有文件被修改过，用git diff可以查看修改内容
